---
title: "Vim Key Binding"
date: "2020-01-01"
---

Session saving  
`:Obsession`  

Toggle undo tree  
`U`  

Ctag sidebar  
`<leader>t`  

Ctag fzf  
`<leader>T`  

zoom  
`<leader>z`  

Quick fix list  
`]q` Next  
`[q` Previous  
`]Q` Last  
`[Q` First  

Location list  
`]l` Next  
`[l` Previous  
`]L` Last  
`[L` First  

Close buffers but keep splits  
`<leader>db`  

Ripgrep  
`<leader>rr`  

Add all ripgrep result to quickfix  
`<alt-a><cr>`
